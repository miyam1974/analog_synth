# NEXUS OSC — UI 機能仕様（SPEC）

本ドキュメントは **現在の UI に表出している機能** をカテゴリ別に網羅する。
実装詳細は [ARCHITECTURE.md](ARCHITECTURE.md)、入門は [README.md](README.md) を参照。

**対象バージョン**: アプリ `0.1.0`（`CMakeLists.txt` の `project(VERSION)`） / ウィンドウ初期サイズ **1080×680**

**範囲**: 画面上のコントロール・表示・演奏入力。**UI に無い機能**（オーディオデバイス設定、VST3、ピッチベンド等）は末尾に一覧する。

---

## UI 構成（上から下）

| 領域 | 内容 |
| ---- | ---- |
| ヘッダー | タイトル「NEXUS OSC」＋サブタイトル「ANALOG SIGNAL ENGINE // POLY 16」（同一行、高さ 28px） |
| マスター行 | ALL OFF / MONO / PRESET / SAVE / SAVE AS / LOAD / RESET / DIFF / MASTER |
| モジュールパネル | OSCILLATOR / MIXER / FILTER / AMPLIFIER / LFO（左→右） |
| SYSTEM フッター | MIDI IN、ステータス行（ヘルプ / MIDI 状態） |
| 仮想キーボード | 画面下部、マウス演奏用 |

モジュール幅（参考）: OSC **21%** / MIXER **17%** / FILTER **23%** / AMP **19%** / 残り LFO。

## 1. グローバル（マスター行）

| UI | 種別 | 説明 | パラメータ / 動作 |
| -- | ---- | ---- | ----------------- |
| **ALL OFF** | ボタン | 全発音の即時停止（パニック） | 全ボイス停止、仮想キーボード状態リセット、MIDI キュークリア |
| **MONO** | トグル | モノフォニック演奏 | ON: 同時 1 音（レガート・GLIDE 向け）。OFF: 16 ボイスポリ |
| **PRESET** | コンボ | プリセット選択 | 内蔵 4 種 + ユーザープリセット名を一覧表示 |
| **SAVE** | ボタン | 読み込み中ユーザープリセットの上書き | 編集後かつユーザープリセット選択時のみ有効。名前は読み取り専用ダイアログ |
| **SAVE AS** | ボタン | 別名保存 | 名前入力ダイアログ → JSON 新規保存。編集後に有効（内蔵プリセット編集中も可） |
| **LOAD** | ボタン | ユーザープリセット読込 | ファイル選択ダイアログ（`*.json`）。DIFF 基準を更新 |
| **RESET** | ボタン | 工場出荷状態へ | INIT 相当。DIFF 基準を更新 |
| **DIFF** | トグル | 基準音色との比較 | ON: 起動時または最後の RESET/LOAD 時の音色を適用し、他コントロールをロック。OFF: 復帰。`D` キーでも切替 |
| **MASTER** | 水平スライダー | マスター音量 | 0〜100% 表示（内部 0.00〜1.00、既定 85%） |

### プリセット

| 種別 | 名称 | 備考 |
| ---- | ---- | ---- |
| 内蔵 | INIT / PAD / BASS / LEAD | 選択即反映 |
| ユーザー | 任意名 | 保存先 `%APPDATA%/NEXUS OSC/Presets/*.json` |

SAVE / SAVE AS / LOAD / RESET / プリセット選択後はベースラインを更新し、未編集なら SAVE / SAVE AS は無効。

## 1b. DIFF 比較モード

| 項目 | 内容 |
| ---- | ---- |
| 基準の記録 | 起動時、**RESET**、**LOAD**、ユーザープリセット **LOAD** 後 |
| DIFF ON | 基準の全パラメータを適用（**MASTER** は保持）。UI は **ALL OFF / MASTER / MIDI IN / DIFF** のみ操作可 |
| DIFF OFF | DIFF ON 前のスナップショットへ復帰（MASTER は DIFF 中の値を維持） |
| キーボード | `D` キーで ON/OFF 切替（モーダルダイアログ表示中は無効） |
| 演奏 | 仮想キーボード・外部 MIDI は DIFF 中も有効 |

| **DET2** | ノブ | ±100 cent | 0 | OSC2 のみ。小数 **2 桁**表示。OSC2 OFF 時は非活性 |
| **DET2 RESET** | ボタン | — | DET2→0 | OSC2 OFF 時は非活性 |

## 6. LFO

2 段独立（**LFO1** / **LFO2**）。各段に RATE / DEPTH、ルーティング 3 種、RATE 連動 LED。

### LFO1

| UI | 種別 | 範囲 | 既定値 | 備考 |
| -- | ---- | ---- | ------ | ---- |
| **RATE** | ノブ | 0.05〜20.0 Hz | 4.00 | |
| **RATE LED** | 表示 | — | — | 位相に連動して点滅（LFO1） |
| **DEPTH** | ノブ | 0.00〜1.00 | 0.30 | |
| **PITCH** | トグル | ON/OFF | OFF | 音程変調 |
| **FILTER** | トグル | ON/OFF | ON | カットオフ変調 |
| **AMP** | トグル | ON/OFF | OFF | 音量変調 |

### LFO2

| UI | 種別 | 範囲 | 既定値 | 備考 |
| -- | ---- | ---- | ------ | ---- |
| **RATE** | ノブ | 0.05〜20.0 Hz | 0.80 | |
| **RATE LED** | 表示 | — | — | LFO2 用 |
| **DEPTH** | ノブ | 0.00〜1.00 | 0.00 | |
| **PITCH** | トグル | ON/OFF | OFF | |
| **FILTER** | トグル | ON/OFF | OFF | |
| **AMP** | トグル | ON/OFF | OFF | |

同一宛先への LFO1 + LFO2 変調は加算。

## 7. SYSTEM

| UI | 種別 | 説明 |
| -- | ---- | ---- |
| **MIDI IN** | コンボ | MIDI 入力デバイス選択 |
| **ステータス行** | ラベル | MIDI 接続状態、またはコントロールホバー時のヘルプ文 |

### MIDI IN の選択肢

| 項目 | 動作 |
| ---- | ---- |
| **All Inputs** | 検出された全 MIDI 入力を同時受信（既定） |
| 個別デバイス名 | 選択した 1 デバイスのみ |
| （デバイスなし） | コンボは "No devices" 表示 |

ステータス例: `MIDI :: ALL INPUTS ACTIVE` / `MIDI :: <デバイス名>` / `MIDI :: NOT CONNECTED`

## 8. 仮想キーボード

| 項目 | 内容 |
| ---- | ---- |
| 操作 | マウスクリックで Note On/Off |
| 音域 | MIDI 36〜96（C1〜C7 付近） |
| 表示 | 押下中キーのハイライト |
| 併用 | 外部 MIDI と同時に `MidiKeyboardState` へマージ |

オーディオデバイスやバッファサイズの UI は無い（OS / JUCE デフォルト出力）。

## 9. 視覚フィードバック（操作以外の UI 表示）

| 機能 | 説明 |
| ---- | ---- |
| **EG 再生ヘッド** | 演奏中、Filter / Amp の EG グラフ上に点（最大 16）が移動 |
| **LFO RATE LED** | LFO1/LFO2 各 1 個。RATE に同期した点滅 |
| **ノブ値ラベル** | 各ノブ下に数値表示（CUT は Hz 整数など） |
| **モジュールパネル枠** | OSC / MIXER / FILTER / AMP / LFO / SYSTEM の区分表示 |

## 11. 演奏仕様（UI から分かる範囲）

| 項目 | 内容 |
| ---- | ---- |
| ポリフォニー | 最大 **16 ボイス**（サブタイトル表示。UI で変更不可） |
| MONO | トグルで切替。レガート時 GLIDE が効く |
| ベロシティ | 外部 MIDI のベロシティを V-A / V-F が反映 |
| ALL OFF | 即時全停止 |
| OSC2 OFF | 波形再クリックで OSC2 出力停止（`osc2Enabled`） |

## 12. セッション永続化

| 項目 | 内容 |
| ---- | ---- |
| 保存先 | `%APPDATA%/NEXUS OSC/session.json` |
| タイミング | アプリ終了時 |

## UI に無い項目（参考）

本節は **画面上にコントロールが無い** ものの整理。  
**実装済み**（動作はするが UI がない）と **未実装**（コードにも無い／スタブのみ）を分けて記載する。

### 実装済み（UI なし）

| カテゴリ | 内容 | 備考 |
| -------- | ---- | ---- |
| オーディオ出力 | Windows デフォルトデバイスへステレオ出力 | JUCE 経由（通常 WASAPI）。**デバイス選択・バッファサイズの UI は無い** |
| MIDI 入力 | 外部 MIDI の受信・演奏 | **MIDI IN** のみ UI あり。チャンネルフィルタ等の UI は無い |
| ポリフォニー | 最大 16 ボイス | サブタイトル表記のみ。**発音数の動的表示 UI は無い** |
| セッション | 終了時に `session.json` へ全パラメータ・プリセット・MIDI・ウィンドウを保存 | UI からの手動保存は無し |
| DIFF 比較 | 基準音色との A/B 切替 | **DIFF** ボタン / `D` キー |

### 未実装

| カテゴリ | 内容 | 備考 |
| -------- | ---- | ---- |
| オーディオ | ASIO 出力、オーディオ設定画面 | ASIO はビルド未リンク。`AudioDeviceSelectorComponent` 未使用 |
| MIDI 演奏 | ピッチベンド、Mod ホイール | `pitchWheelMoved` / `controllerMoved` は空実装 |
| プラグイン | VST3 / CLAP | Standalone EXE のみ |
| FX | コーラス / ディレイ / リバーブ | 信号チェーンに無し |
| DSP | SmoothedValue、実効 Cutoff Hz 表示 | パラメータは atomic 直読み。モジュレーション後の Hz 表示なし |
| その他 | アルペジエータ、MPE、ポリフォニー数の動的表示 | 将来候補 |

詳細は [ARCHITECTURE.md](ARCHITECTURE.md) の「現状の制約と今後の拡張」を参照。
